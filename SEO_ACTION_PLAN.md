## 3. Internal Linking Strategy

### Hub-and-Spoke Model
```
Homepage (/)
├── /cleaning-services-montreal  ← Services hub
│   ├── /deep-cleaning-montreal
│   ├── /move-in-out-cleaning-montreal
│   ├── /airbnb-cleaning-montreal
│   ├── /post-renovation-cleaning-montreal
│   └── /recurring-cleaning-montreal
├── /prix-nettoyage-montreal     ← Pricing
├── /quote                       ← Conversion
└── Area pages (spoke)
    ├── /ndg-cleaning-services
    ├── /plateau-cleaning-services
    ├── /westmount-cleaning-services
    ├── /laval-cleaning-services
    ├── /brossard-cleaning-services
    └── /west-island-cleaning-services
```

### Mandatory Links on Every Page
- Nav: Home, Services, Pricing, Areas, Get a Quote
- Footer: all service pages, all area pages, Pricing, Get a Quote
- CTA banner: links to `/quote` AND BookingKoala booking widget

### Service Page → Area Cross-Links
Each service page should link to the top 3 area pages in a "Where we offer this service" section.

### Area Page → Service Cross-Links
Each area page should link to all 6 service pages with area-specific anchor text.
Example: `/ndg-cleaning-services` → "deep cleaning in NDG" → `/deep-cleaning-montreal`

### Pricing Page
- Link to every service page with the price listed
- Link to `/quote` from every price row ("Get exact quote")

### Quote Page
- Link to `/prix-nettoyage-montreal` for pricing reference (already done in sidebar)
- Link to all area pages in the "areas we serve" section (already done)
